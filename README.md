# Secure Vault — End-to-End Encrypted File Storage

A **full-stack secure file storage application** with end-to-end encryption.  
Users can upload, store, and retrieve files securely. All files are encrypted before storage, ensuring complete data privacy.

---

## Features

- User authentication & authorization
- File upload & download
- End-to-end encryption (AES / cryptography module)
- File management: delete, rename, search
- Full-stack: backend + frontend interface
- Secure database integration for file metadata

---

## Installation

1. Install dependencies:
```bash
pip install -r requirements.txt
````

2. Run backend:

```bash
python app.py
```

3. Open frontend in browser and connect to backend server.
